mtls_t mtls_connect ( int  fd,
int  calen,
char *  ca,
int *  errno 
)

TLS client create a TLS connection.

This function is called on the client side and initiates an TLS/TLS handshake with a server. When this function is called, the underlying communication channel has already been set up. This function is called after TCP 3-way handshak finished.

Parameters
fdThe fd number for a connected TCP socket.
calenthe string length of ca. 0=do not verify server's certificate.
capointer to the CA certificate string, used to verify server's certificate.
errnoreturn the errno if mtls_connect return NULL.
Returns
return the TLS context pointer on success or NULL for fail.

void mtls_set_cert ( const char *  cert_pem,
const char *  private_key_pem,
const char *  verify_ca 
)

Used by the TLS server. Set the certificate and private key for a TLS server.

This function is called on the server side to set it's certifact and private key. It must be called before mtls_accept. These two arguments must be global string buffer, library will not create a copy for them.

Parameters
cert_pemPoint to the certificate string in PEM format.
private_key_pemPoint to the private key string in PEM format.
verify_caverify client certificate.
